# 多De que va?

* Son dos visualizadores de codigo a la hora de escribirlo los resultados se actualizan al instante porlo que puede ser bastante util

# 多Donde puedo acceder ha ellos?

* Visualizadores de HTML [html.maucode.com](https://html.maucode.com)
* Visualizadores de MARKDOWN [md.maucode.com](https://md.maucode.com)
* Visualizadores de CODIGO [md.maucode.com](https://maubot.maucode.com/compilador)

# 多Como se ven?

* Se ve precioso
* En el editor de Markdown puedes descargarte un archivo `.pdf` y se puede poner en modo VIM

# 多Algo mas?

* En la web de [Maubor](https://maubot.maucode.com) se puede ver un [editor](https://maubot.maucode.com/compilador) de programacion en el que puedes compilar, descargar tu codigo en mas de 20 lenguages de programacion.
